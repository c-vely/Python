[통신]
- 풀 듀플렉스 : ex) USART, UART
- 하프 듀플렉스 : ex) usb

- CAN : 특정 영역 통신망 ex) 차안 

* 라즈베리에서는 아래 2개 사용
- SPI : 풀 디플렉스, Data In(MISO) / Data Out(MOSI) / CLK
- I2C : 하프 디플렉스, Data / CLK
